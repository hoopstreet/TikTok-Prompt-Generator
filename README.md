## Usage

1. Enter **Product Title** (required)
2. Enter **About This Product** (required)
3. Enter **Product Description** (required)
4. (Optional) Add **Image URL** from TikTok Shop
5. Click **Generate Prompt**
6. Copy the output for your TikTok Affiliate video

## Output Format

Each generation includes:
- **positive_prompt** - Shot-by-shot visual descriptions with Taglish dialogue
- **negative_prompt** - Quality exclusion list (1:1 mapping)
- **final_title** - SEO title with 5 hashtags
- **4 Cards** - Product Info, Video Brief, Analysis, Storyboard
